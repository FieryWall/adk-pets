# ADK Capstone Project

## Dependencies:
- Python 3.12+
- Google ADK 1.18.0
- Google GenAI 1.49.0

## Installation:
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd <project-directory>
   ```
3. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```
4. Set up Google ADK credentials as per the [Google ADK documentation](https://google.github.io/adk-docs/get-started/python/#set-your-api-key). Your environment should have the `GOOGLE_API_KEY` variable set.

## How to run Python script:
1. Ensure you have installed Python:
   ```bash
   python --version
   ```
   or 
   ```bash
   python3 --version
   ```
2. Navigate to the project directory if not already there:
   ```bash
   cd <project-directory>
   ```
3. Execute the script using Python:
   ```bash
   python main.py
   ```
## How to run Jupyter Notebook locally:
1. Start Jupyter Notebook server:
   ```bash
   jupyter lab --no-browser
   ```
2. Open the `dima.ipynb` notebook and connect to the running server.
